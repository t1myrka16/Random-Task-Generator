# Random Task Generator

**Автор:** Хайруллин Тимур Рустамович

## Описание

Простое приложение для генерации случайных задач с возможностью:
- Выбора случайной задачи из предопределённого списка
- Добавления новых задач с указанием типа (учёба, спорт, работа)
- Фильтрации истории по типу задачи
- Сохранения истории в JSON-файл
- Загрузки истории при запуске

## Технологии

- Python 3
- tkinter (встроенный GUI)
- random
- json
- os

## Установка и запуск

1. Убедитесь, что установлен Python 3.6 или выше
2. Скачайте файл `task_generator.py`
3. Запустите командой:
   ```bash
   python task_generator.py
